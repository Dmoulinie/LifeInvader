Pour le projet Instagram (DevWeb)

Sujet : Micro-service en Python 3

Fonctionnalités :
- Gère les requetes, les erreurs et la connexion à 
une base de données MongoDB
- Charge, modifie et colore différents formats d'images

Librairies : 
- Flask (service de serveur)
- Pymongo (connection à un client mongodb)
- Grifs (ajouter de larges fichiers dans mongodb)
- OS (charger les fichiers locaux)


Instructions : Lancer le main.py de test pour voir si les 
fonctionnalités de base ne retournent pas une erreur
